Windows

  1. 将生成的 mydll.dll 文件和所有依赖的 DLL 文件（如 opencv_world.dll、nvinfer.dll 等）复制到 C# 项目的输出目录（通常是 bin\Debug 或 bin\Release）。
  2. 或者，将这些 DLL 文件的路径添加到系统的 PATH 环境变量中。

Linux

  1. 将生成的 libmydll.so 文件和所有依赖的共享库文件（如 libopencv_world.so、libnvinfer.so 等）复制到 C# 项目的输出目录。
  2. 或者，将这些共享库文件的路径添加到 LD_LIBRARY_PATH 环境变量中。
